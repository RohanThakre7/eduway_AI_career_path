# EduWay – Frontend Landing Page

An interactive, premium-designed landing page for the AI-Powered Personalized Learning & Career Guidance Platform. This page connects users directly to the AI Engine for diagnostic career surveys and roadmaps.

## 🌐 Live Access
- **Landing Page (Frontend):** [https://eduway-ai-career-path.vercel.app](https://eduway-ai-career-path.vercel.app)
- **AI Career Planner Engine (Backend):** [https://eduway.streamlit.app/](https://eduway.streamlit.app/)

## 🚀 How to Run Locally

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Steps
1. **Clone the repository:**
   ```bash
   git clone https://github.com/RohanThakre7/eduway_AI_career_path.git
   cd eduway_AI_career_path
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the local development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🛠️ Tech Stack
- **Framework:** React + Vite
- **Styling:** Tailwind CSS + Framer Motion (for premium animations)
- **Icons:** Lucide React
- **Deployment:** Vercel
